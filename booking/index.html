Installation readiness

For Codex / OpenClaw / Hermes installation sessions.

Dedicated Mac

Use a dedicated Mac, not a personal or primary work machine. Keep personal accounts and identifying device names off the setup.

Local models

For privacy- or security-sensitive data, systems can be designed to use local models exclusively when the client is willing to secure hardware capable enough for the open-source models the work requires.

Remote access

NoMachine can speed up administration. For live observation from another device, a NoMachine Enterprise Desktop license may be needed.

Availability

Be physically present for the first hour of installation work and reachable by phone afterward during the remaining setup window.
