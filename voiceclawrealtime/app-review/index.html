VoiceClaw Realtime

App Review setup code.

This page is intended for App Store Review. Scan the setup code in VoiceClaw Realtime to load review-only settings for testing the app's core flow.

How to test

  1. Install and launch the selected review build of VoiceClaw Realtime.
  2. Open Settings, then the Setup tab.
  3. Tap Scan QR and scan the setup code on this page.
  4. Open Diagnostics to confirm setup, then return to Live and start a voice session.

This code contains review-only credentials and a temporary review tunnel. These settings are provided only for App Store Review and will be revoked after review is complete.
