Getting started

Setup checklist

  1. Install VoiceClaw Realtime on iPhone from the App Store or TestFlight.
  2. Open Settings in the iPhone app and add either an OpenAI API key or pair with VoiceClaw Companion to use OpenClaw OAuth through your ChatGPT Plus/Pro subscription.
  3. Download the macOS Companion app from the VoiceClaw Companion Releases page.
  4. If you use OpenClaw on your Mac, install OpenClaw and point VoiceClaw Companion to your OpenClaw installation.
  5. If you want private iPhone-to-Mac access, install and sign in to Tailscale on both devices. If you want Apple Watch access when the iPhone is not nearby, configure an optional HTTPS tunnel.
  6. Use the QR code in VoiceClaw Companion to sync setup settings to iPhone, then use Sync to iPhone in the Apple Watch app to sync watch settings.

OpenAI and subscription access

VoiceClaw can use an OpenAI API key entered by the user. It can also use OpenClaw OAuth through VoiceClaw Companion when OpenClaw is installed and configured on the user's Mac.

VoiceClaw Companion

The Mac Companion installs and runs the local bridge used for pairing, diagnostics, Tailscale Serve configuration, optional HTTPS tunnel settings, and OpenClaw access.

Apple Watch

The watchOS app can use Direct GPT-Realtime-2, GPT-5.5 Instant, iPhone relay, or an optional HTTPS tunnel, depending on your settings and network availability.

Billing

Subscriptions are handled by Apple through the App Store. To manage or cancel a subscription, open Settings on your device, tap your Apple ID, then Subscriptions.

Troubleshooting

Common issues

No voice output
Check Diagnostics in the iPhone app, confirm microphone permission, confirm speaker output, and make sure a valid OpenAI API key or OpenClaw OAuth route is configured.
Cannot reach OpenClaw
Confirm VoiceClaw Companion is running on your Mac, OpenClaw is installed, the OpenClaw install path is correct, and either Tailscale or an optional HTTPS tunnel is configured.
Apple Watch cannot reach the Mac
Apple Watch cannot use private Tailscale networking by itself. Use iPhone relay when the iPhone is nearby, or configure an optional HTTPS tunnel if you want watch access away from iPhone.
Subscription or purchase issue
Use Restore Purchases in the app. If App Store billing still does not work, contact Apple Support for billing and email ben@benbadejo.com for app support.
